# 외장 모듈 목록

IMCAT UI는 22개의 외장 모듈을 제공합니다. 모든 모듈은 `IMCAT.use()`로 동적 로딩됩니다.

## 모듈 로딩 방법

```javascript
// 단일 모듈 로드
const Overlays = await IMCAT.use('overlays');

// 모듈 사용
const modal = new Overlays.Modal({ title: '제목' });
modal.show();
```

---

## 카테고리별 모듈

### Overlays & Dialogs

| 모듈 | 컴포넌트 | 설명 |
|------|----------|------|
| `overlays` | Modal, Drawer, Offcanvas | 오버레이 다이얼로그 |

```javascript
const Overlays = await IMCAT.use('overlays');

// Modal
const modal = new Overlays.Modal({
  title: '알림',
  content: '<p>작업이 완료되었습니다.</p>',
  size: 'md', // sm, md, lg, xl
  buttons: [
    { text: '확인', variant: 'primary', action: () => modal.hide() }
  ]
});
modal.show();

// Drawer
const drawer = new Overlays.Drawer({
  title: '메뉴',
  position: 'left', // left, right, top, bottom
  content: '<nav>...</nav>'
});
drawer.show();

// Offcanvas
const offcanvas = new Overlays.Offcanvas({
  title: '설정',
  position: 'right',
  width: '400px'
});
```

### Feedback

| 모듈 | 컴포넌트 | 설명 |
|------|----------|------|
| `feedback` | Toast, Notification | 사용자 피드백 |

```javascript
const Feedback = await IMCAT.use('feedback');

// Toast (간단한 메시지)
Feedback.Toast.success('저장되었습니다');
Feedback.Toast.error('오류가 발생했습니다');
Feedback.Toast.warning('주의!');
Feedback.Toast.info('알림');

// Notification (상세 알림)
Feedback.Notification.show({
  title: '새 메시지',
  message: '새로운 메시지가 도착했습니다.',
  type: 'info',
  duration: 5000,
  actions: [
    { text: '보기', onClick: () => openMessage() }
  ]
});
```

### Dropdown

| 모듈 | 컴포넌트 | 설명 |
|------|----------|------|
| `dropdown` | Dropdown | 드롭다운 메뉴 |

```javascript
const { Dropdown } = await IMCAT.use('dropdown');

const dropdown = new Dropdown('#trigger-btn', {
  items: [

    { label: '복사', icon: 'content_copy', onClick: () => copy() },

  ],
  position: 'bottom', // top, bottom, left, right
  align: 'start'      // start, center, end
});
```

### Navigation

| 모듈 | 컴포넌트 | 설명 |
|------|----------|------|
| `navigation` | Tabs, Accordion, TreeView | 네비게이션 컴포넌트 |

```javascript
const Navigation = await IMCAT.use('navigation');

// Tabs
const tabs = new Navigation.Tabs('#tabs-container', {
  tabs: [
    { id: 'tab1', label: '탭 1', content: '<p>내용 1</p>' },
    { id: 'tab2', label: '탭 2', content: '<p>내용 2</p>' }
  ],
  variant: 'default' // default, pills, cards
});

// Accordion
const accordion = new Navigation.Accordion('#accordion', {
  items: [
    { title: '섹션 1', content: '<p>내용</p>' },
    { title: '섹션 2', content: '<p>내용</p>' }
  ],
  multiple: false // 동시 여러 개 열기 허용
});
```

### Pickers

| 모듈 | 컴포넌트 | 설명 |
|------|----------|------|
| `pickers` | DatePicker, TimePicker, ColorPicker | 선택기 |

```javascript
const Pickers = await IMCAT.use('pickers');

// DatePicker
const datePicker = new Pickers.DatePicker('#date-input', {
  format: 'YYYY-MM-DD',
  minDate: new Date(),
  onChange: (date) => console.log(date)
});

// TimePicker
const timePicker = new Pickers.TimePicker('#time-input', {
  format: '24h', // 12h, 24h
  step: 30       // 분 단위
});

// ColorPicker
const colorPicker = new Pickers.ColorPicker('#color-input', {
  defaultColor: '#3B82F6',
  presets: ['#EF4444', '#10B981', '#3B82F6']
});
```

### Selectors

| 모듈 | 컴포넌트 | 설명 |
|------|----------|------|
| `selectors` | Autocomplete, MultiSelect, Transfer | 선택 컴포넌트 |

```javascript
const Selectors = await IMCAT.use('selectors');

// Autocomplete
const autocomplete = new Selectors.Autocomplete('#search', {
  source: async (query) => {
    const res = await fetch(`/api/search?q=${query}`);
    return res.json();
  },
  onSelect: (item) => console.log(item)
});

// MultiSelect
const multiSelect = new Selectors.MultiSelect('#tags', {
  options: [
    { value: 1, label: 'Option 1' },
    { value: 2, label: 'Option 2' }
  ],
  max: 5 // 최대 선택 수
});

// Transfer
const transfer = new Selectors.Transfer('#transfer', {
  sourceTitle: '선택 가능',
  targetTitle: '선택됨',
  items: [...]
});
```

### Live Status

| 모듈 | 컴포넌트 | 설명 |
|------|----------|------|

```javascript
  rows: 3,
  avatar: true
});
skeleton.show();
skeleton.hide();
```

```javascript
  orientation: 'horizontal' // horizontal, vertical

stepper.next();
stepper.prev();
stepper.goTo(2);
```

## 메모리 관리

모든 모듈은 `destroy()` 메서드를 제공합니다.

```javascript
// 수동 정리
modal.destroy();

// 자동 정리 (페이지 이동 시)
IMCAT.view.registerInstance(modal);
```

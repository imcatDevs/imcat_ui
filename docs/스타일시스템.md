# 스타일 시스템

IMCAT UI는 SCSS 기반의 모던 디자인 시스템을 제공합니다.

## 파일 구조

```
src/styles/
├── abstracts/           # 변수, 믹스인, 함수
│   ├── _variables.scss  # 디자인 토큰
│   ├── _mixins.scss     # 재사용 믹스인
│   ├── _functions.scss  # 유틸리티 함수
│   ├── _themes.scss     # 테마 시스템
│   └── _fonts.scss      # 폰트 정의
├── base/                # 기본 스타일
│   ├── _reset.scss      # CSS 리셋
│   └── _typography.scss # 타이포그래피
├── modules/             # 모듈 스타일 (15개)
│   ├── _overlays.scss
│   ├── _feedback.scss
│   ├── _dropdown.scss
│   └── ...
└── imcat-ui.scss        # 메인 엔트리
```

## 디자인 토큰

### 색상 시스템

```scss
// Primary 색상
$primary: #3B82F6;
$primary-hover: #2563EB;
$primary-active: #1D4ED8;

// Semantic 색상
$success: #10B981;
$warning: #F59E0B;
$error: #EF4444;
$info: #3B82F6;
$danger: #DC2626;

// 배경 색상 (라이트)
$bg-primary: #FFFFFF;
$bg-secondary: #F9FAFB;
$bg-tertiary: #F3F4F6;

// 텍스트 색상
$text-primary: #111827;
$text-secondary: #6B7280;
$text-tertiary: #9CA3AF;
$text-disabled: #D1D5DB;

// 테두리
$border-color: #E5E7EB;
$border-strong: #D1D5DB;
```

### CSS 변수

테마 시스템은 CSS 변수를 사용합니다:

```css
:root {
  --primary: #3B82F6;
  --bg-primary: #FFFFFF;
  --text-primary: #111827;
  --border-color: #E5E7EB;
}

[data-theme="dark"] {
  --primary: #60A5FA;
  --bg-primary: #111827;
  --text-primary: #F9FAFB;
  --border-color: #374151;
}
```

### 사용 방법

```scss
.my-component {
  background: var(--bg-primary, $bg-primary);
  color: var(--text-primary, $text-primary);
  border: 1px solid var(--border-color, $border-color);
}
```

## z-index 계층

```scss
// 계층 구조
$z-base: 1-11;           // 내부 요소
$z-local: 100;           // 로컬 오버레이
$z-dropdown: 1000;       // 드롭다운
$z-overlay: 1050;        // 오버레이 백드롭
$z-overlay-content: 1051;// 모달, 드로어
$z-selector: 1055;       // 셀렉터 드롭다운
$z-picker: 1060;         // 피커
$z-popover: 1070;        // 팝오버
$z-tooltip: 1080;        // 툴팁
$z-toast: 1100;          // 토스트, 알림
$z-top: 2000;            // 라이트박스
```

## BEM 네이밍 규칙

모든 클래스는 BEM 패턴을 따릅니다:

```scss
// Block
.modal { }

// Element
.modal__header { }
.modal__body { }
.modal__footer { }

// Modifier
.modal--sm { }
.modal--lg { }

// State (is-* 패턴)
.modal.is-visible { }
.btn.is-active { }
.input.is-disabled { }
```

### 상태 클래스

```scss
.is-visible    // 표시 상태
.is-active     // 활성 상태
.is-selected   // 선택 상태
.is-disabled   // 비활성 상태
.is-error      // 에러 상태
.is-loading    // 로딩 상태
.is-open       // 열림 상태
```

## 다크 모드

### HTML 속성

```html
<html data-theme="dark">
```

### JavaScript 제어

```javascript
// 테마 설정
document.documentElement.setAttribute('data-theme', 'dark');

// IMCAT API
IMCAT.theme.set('dark');
IMCAT.theme.toggle();
```

### 조건부 스타일

```scss
.my-component {
  background: var(--bg-primary);
  
  [data-theme="dark"] & {
    // 다크 모드 전용 스타일
  }
}
```

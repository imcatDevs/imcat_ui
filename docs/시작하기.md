# 시작하기

## 설치

### NPM

```bash
npm install @imcat-ckim/catui
```

### CDN

```html
<link rel="stylesheet" href="dist/imcat-ui.css">
<script type="module" src="dist/imcat-ui.min.js"></script>
```

## 기본 설정

### HTML 구조

```html
<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My App</title>
  <link rel="stylesheet" href="dist/imcat-ui.css">
</head>
<body>
  <!-- 네비게이션 -->
  <nav>
    <a catui-href="views/home.html">홈</a>
    <a catui-href="views/about.html">소개</a>
  </nav>
  
  <!-- 콘텐츠 영역 -->
  <main id="app-content"></main>
  
  <script type="module">
    import IMCAT from './dist/imcat-ui.min.js';
    
    // 초기 페이지 로드
    IMCAT.view.navigate('views/home.html');
  </script>
</body>
</html>
```

## 핵심 개념

### 1. DOM 선택 및 조작

```javascript
// 요소 선택 (jQuery 스타일)
const app = IMCAT('#app');
const buttons = IMCAT('.btn');

// 체이닝
IMCAT('#app')
  .addClass('active')
  .removeClass('hidden')
  .text('Hello World')
  .on('click', () => console.log('clicked'));

// 요소 순회
IMCAT('.nav-link').each((el, index) => {
  console.log(el, index);
});
```

### 2. SPA 라우터

```javascript
// 페이지 이동
IMCAT.view.navigate('views/dashboard.html');

// 쿼리 파라미터 가져오기
const params = IMCAT.view.params();
console.log(params.id);

// 라이프사이클 훅
IMCAT.view.beforeLoad((path) => {
  console.log('로딩 전:', path);
});

IMCAT.view.afterLoad((path) => {
  console.log('로딩 후:', path);
});
```

### 3. 모듈 로딩

```javascript
// 모듈 동적 로드
const Overlays = await IMCAT.use('overlays');
const Feedback = await IMCAT.use('feedback');

// 모듈 사용
const modal = new Overlays.Modal({
  title: '알림',
  content: '작업이 완료되었습니다.'
});
modal.show();
```

### 4. 이벤트 버스

```javascript
// 이벤트 구독
IMCAT.on('user:login', (data) => {
  console.log('로그인:', data.username);
});

// 이벤트 발행
IMCAT.emit('user:login', { username: 'john' });

// 일회성 구독
IMCAT.once('app:ready', () => {
  console.log('앱 준비 완료');
});

// 구독 취소
IMCAT.off('user:login', handler);
```

### 5. 상태 관리

```javascript
// 로컬 상태 (컴포넌트용)
const state = IMCAT.state.create({ count: 0 });

state.watch('count', (newVal, oldVal) => {
  console.log(`${oldVal} -> ${newVal}`);
});

state.count++; // 자동 감지

// 전역 상태 (앱 전체 공유)
const userStore = IMCAT.globalState.use('user', {
  isLoggedIn: false,
  name: ''
});

userStore.isLoggedIn = true;
```

### 6. 스토리지

```javascript
// 저장 (TTL 지원)
IMCAT.storage.set('token', 'abc123', { expires: 3600 }); // 1시간

// 조회
const token = IMCAT.storage.get('token');

// 삭제
IMCAT.storage.remove('token');

// 세션 스토리지
IMCAT.storage.set('temp', 'data', { storage: 'session' });
```

## 단축 API

### 다이얼로그

```javascript
// 알림
await IMCAT.alert('저장되었습니다');

// 확인
const confirmed = await IMCAT.confirm('삭제하시겠습니까?');
if (confirmed) {
  // 삭제 로직
}

// 입력
const name = await IMCAT.prompt('이름을 입력하세요');
```

### 토스트

```javascript
IMCAT.toast.success('저장 완료');
IMCAT.toast.error('오류가 발생했습니다');
IMCAT.toast.warning('주의가 필요합니다');
IMCAT.toast.info('새로운 알림이 있습니다');
```

### 알림

```javascript
IMCAT.notify.success('성공', '작업이 완료되었습니다');
IMCAT.notify.error('오류', '처리 중 문제가 발생했습니다', {
  duration: 0,  // 자동 닫힘 안함
  actions: [
    { text: '재시도', onClick: () => retry() }
  ]
});
```

### 테마

```javascript
// 테마 전환
IMCAT.theme.toggle();         // 토글
IMCAT.theme.set('dark');      // 다크 모드
IMCAT.theme.set('light');     // 라이트 모드
IMCAT.theme.set('system');    // 시스템 설정 따름

// 현재 테마
console.log(IMCAT.theme.current);
```

## 유틸리티

```javascript
// 타입 체크
IMCAT.isString('hello');  // true
IMCAT.isArray([1, 2]);    // true
IMCAT.isObject({});       // true

// ID 생성
const id = IMCAT.randomId('modal'); // 'modal-a1b2c3'

// 디바운스/스로틀
const debouncedSearch = IMCAT.debounce(search, 300);
const throttledScroll = IMCAT.throttle(onScroll, 100);

// 객체 병합/복사
const merged = IMCAT.extend({}, obj1, obj2);
const cloned = IMCAT.clone(obj);

// HTML 이스케이프 (XSS 방어)
const safe = IMCAT.escape('<script>alert("xss")</script>');
```

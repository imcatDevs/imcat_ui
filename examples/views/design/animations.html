Animation System

Animations bring life to the interface and provide visual feedback for user interactions.

Transition Properties

Property Duration Timing Function Use Case
all 0.2s ease General interactions (hover, focus)
transform 0.3s ease-out Movement, scaling
opacity 0.15s linear Fade in/out
color 0.2s ease Text color changes
Example
/* CSS Transition */
.button {
  transition: all 0.2s ease;
}

.button:hover {
  transform: translateY(-2px);
  box-shadow: 0 4px 8px rgba(0,0,0,0.1);
}

Timing Functions (Easing)

Function Description Use Case
linear Constant speed Loading indicators, simple fades
ease Slow start, fast middle, slow end General purpose
ease-in Slow start, then accelerate Exit animations
ease-out Fast start, then decelerate Enter animations
ease-in-out Slow start and end Smooth transitions

Animation Best Practices

  • check_circle Keep it subtle: Animations should enhance, not distract (200-300ms typical)
  • check_circle Purpose: Every animation should have a clear purpose
  • check_circle Performance: Use transform and opacity for best performance
  • check_circle Accessibility: Respect prefers-reduced-motion
  • warning Avoid: Animations longer than 500ms
  • warning Avoid: Animating layout properties (width, height, margin)
Accessibility
/* Respect user preferences */
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}

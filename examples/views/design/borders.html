Border Best Practices

  • check_circle Consistency: Use consistent border widths (1px for most cases)
  • check_circle Subtle colors: Use neutral colors for most borders
  • check_circle Radius: Match border radius with design language (4-8px typical)
  • warning Avoid: Too many different border styles on one page
  • warning Avoid: Overly thick borders that distract from content
Code Example
/* CSS */
.card {
  border: 1px solid var(--border-color);
  border-radius: 8px;
}

/* Accent border */
.card--accent {
  border: 2px solid var(--primary-color);
}

/* HTML */
<div class="border border-primary rounded">
  Content
</div>

Spacing System

IMCAT UI uses a consistent spacing scale based on Material Design's 4px/8px grid system.

Usage Guidelines

  • check_circle Consistency: Use the spacing scale consistently throughout the design
  • check_circle Hierarchy: Larger spacing creates visual hierarchy and grouping
  • check_circle Breathing room: Adequate spacing improves readability
  • warning Avoid: Creating custom spacing values outside the scale
Code Example
<!-- Margin -->
<div class="mt-4 mb-3">Content</div>

<!-- Padding -->
<div class="p-4">Content</div>

<!-- Flexbox gap -->
<div class="d-flex gap-3">
  <div>Item 1</div>
  <div>Item 2</div>
</div>

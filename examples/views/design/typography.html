Responsive Typography

Typography scales automatically based on viewport size for optimal readability on all devices.

Breakpoint Base Font Size H1 Size
Mobile (<576px) 14px 2rem (32px)
Tablet (≥576px) 15px 2.25rem (36px)
Desktop (≥768px) 16px 2.5rem (40px)

Typography Best Practices

  • check_circle Use semantic HTML: Use proper heading hierarchy (h1-h6)
  • check_circle Maintain contrast: Ensure text has sufficient contrast ratio (WCAG AA: 4.5:1)
  • check_circle Line length: Keep line length between 50-75 characters for optimal readability
  • check_circle Line height: Use 1.5-1.6 for body text, 1.2-1.3 for headings
  • warning Avoid: Using all caps for long text passages
  • warning Avoid: Too many font weights or sizes in one design

Boxed Layout Overview

Boxed Layout constrains the layout to a maximum width, creating a centered container with margins on either side. This provides a more focused and organized appearance, especially on larger screens.

info Note: To enable Boxed Layout, add the .layout-boxed class to the <body> tag.
Features
  • Maximum width: 1400px (customizable)
  • Centered container with side margins
  • Box shadow for depth effect
  • Optimal reading width for content
  • Professional and modern appearance
Implementation
<body class="layout-boxed">
  <div class="wrapper">
    <!-- Sidebar -->
    <div class="sidenav-menu">...</div>
    
    <!-- Content -->
    <div class="content-page">
      <div class="main-content">...</div>
    </div>
  </div>
</body>

SCSS Configuration

// Boxed Layout Variables
.layout-boxed {
  background: var(--bg-secondary);
  min-height: 100vh;
  
  .wrapper {
    max-width: 1400px;  // Customize this value
    margin: 0 auto;
    box-shadow: 0 0 35px 0 rgba(154, 161, 171, 0.15);
    background: var(--bg-primary);
  }
  
  .sidenav-menu {
    position: fixed;
    left: auto;
    max-width: 260px;
    
    @media (min-width: 1400px) {
      left: calc((100vw - 1400px) / 2);
    }
  }
  
  .content-page {
    margin-left: 260px;
    min-height: 100vh;
  }
}
Customization Options
Property Default Value Description
max-width 1400px Maximum width of the boxed container
box-shadow 0 0 35px rgba(...) Shadow effect around the container
background var(--bg-secondary) Background color outside the box
sidebar-width 260px Width of the sidebar

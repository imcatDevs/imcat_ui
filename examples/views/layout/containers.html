Container Types

Containers are the most basic layout element and are required when using the grid system. Choose from a responsive, fixed-width container or a fluid-width container.

Responsive Container

Use .container for a responsive fixed width container. The width adjusts based on the viewport size.

.container - Max-width changes at each breakpoint
Fluid Container

Use .container-fluid for a full width container, spanning the entire width of the viewport.

.container-fluid - Always 100% width

Responsive Breakpoint Containers

Use breakpoint-specific container classes for fluid width until the specified breakpoint.

.container-sm

100% wide until small breakpoint (≥576px)

100% width until sm breakpoint, then fixed
.container-md

100% wide until medium breakpoint (≥768px)

100% width until md breakpoint, then fixed
.container-lg

100% wide until large breakpoint (≥1024px)

100% width until lg breakpoint, then fixed
.container-xl

100% wide until extra large breakpoint (≥1280px)

100% width until xl breakpoint, then fixed
.container-xxl

100% wide until extra extra large breakpoint (≥1536px)

100% width until xxl breakpoint, then fixed

Usage Examples

Basic Container with Grid
<div class="container">
  <div class="row">
    <div class="col-md-6">Column 1</div>
    <div class="col-md-6">Column 2</div>
  </div>
</div>
Fluid Container
<div class="container-fluid">
  <!-- Full width content -->
</div>
Responsive Container
<div class="container-lg">
  <!-- 100% width until lg breakpoint -->
</div>

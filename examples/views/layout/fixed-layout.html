Fixed Layout Overview

Fixed Layout keeps the sidebar and topbar fixed in position while scrolling. This provides a consistent navigation experience for users.

info Note: To enable Fixed Layout, add the .layout-fixed class to the <body> tag.
Features
  • Sidebar remains fixed during scroll
  • Topbar stays at the top of the viewport
  • Content scrolls independently
  • Optimal for content-heavy applications
  • Responsive on mobile devices
Implementation
<body class="layout-fixed">
  <div class="wrapper">
    <!-- Sidebar -->
    <div class="sidenav-menu">...</div>
    
    <!-- Content -->
    <div class="content-page">
      <!-- Topbar -->
      <div class="app-topbar">...</div>
      
      <!-- Main Content -->
      <div class="main-content">...</div>
    </div>
  </div>
</body>

SCSS Variables

// Fixed Layout Variables
.layout-fixed {
  .sidenav-menu {
    position: fixed;
    top: 0;
    left: 0;
    bottom: 0;
    z-index: 1000;
  }
  
  .app-topbar {
    position: fixed;
    top: 0;
    right: 0;
    left: 260px;  // sidebar width
    z-index: 999;
  }
  
  .content-page {
    margin-left: 260px;
  }
  
  .main-content {
    padding-top: 70px;  // topbar height
  }
}

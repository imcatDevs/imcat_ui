Extra extra large .col-xxl- ≥1536px

Responsive Display Utilities

Use display utilities to show or hide elements at different breakpoints.

Hide on Mobile
info This alert is hidden on mobile (smaller than md breakpoint)
Show Only on Mobile
smartphone This alert is only visible on mobile devices
Display Utility Classes
Class Description
.d-none Hide on all screen sizes
.d-block Display as block on all screen sizes
.d-sm-none Hide on sm and larger screens
.d-md-block Display as block on md and larger screens
.d-lg-flex Display as flex on lg and larger screens
.d-xl-inline-block Display as inline-block on xl and larger screens

Responsive Spacing

Spacing utilities can be applied responsively using breakpoint prefixes.

Examples
.mb-2 .mb-md-4 - Small margin on mobile, larger on medium+
.p-2 .p-lg-4 - Small padding on mobile, larger on large+
Spacing Classes
.{property}{sides}-{size}
.{property}{sides}-{breakpoint}-{size}

Properties: m (margin), p (padding)
Sides: t (top), b (bottom), s (start), e (end), x (horizontal), y (vertical)
Sizes: 0, 1, 2, 3, 4, 5, auto
Breakpoints: sm, md, lg, xl, xxl

Responsive Design Best Practices

  • check_circle Mobile First: Design for mobile devices first, then enhance for larger screens
  • check_circle Touch Targets: Ensure buttons and links are at least 44x44px on mobile
  • check_circle Readable Text: Use font sizes of at least 16px for body text on mobile
  • check_circle Test Often: Regularly test your layouts on different devices and screen sizes
  • check_circle Performance: Optimize images and assets for different screen densities

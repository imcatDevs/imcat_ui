Sidebar Layout Overview

IMCAT UI currently uses a left sidebar layout as demonstrated in this documentation interface. The sidebar can be minimized on desktop and hidden on mobile devices.

info Current Layout: You are viewing the left sidebar layout with collapsible menu.

Left Sidebar Layout (Current)

Features
  • Fixed Position: Sidebar remains fixed while scrolling content
  • Collapsible: Can be minimized to icon-only mode on desktop
  • Responsive: Hides off-canvas on mobile, accessible via toggle button
  • Scrollable: Independent scroll for menu items
Structure
<div class="wrapper">
  <!-- Sidebar -->
  <div class="sidenav-menu">
    <div class="scrollbar">
      <ul class="side-nav">
        <!-- Menu items -->
      </ul>
    </div>
  </div>

  <!-- Content -->
  <div class="content-page">
    <!-- Topbar -->
    <header class="app-topbar"></header>
    
    <!-- Main Content -->
    <div class="content"></div>
  </div>
</div>

Mini Sidebar (Icon Only)

Minimized sidebar showing only icons. Click the toggle button in the current layout to see this in action.

Features
  • Space Efficient: Provides more room for main content
  • Icon Navigation: Menu items represented by icons only
  • Expandable: Can expand to full width on hover or click
  • Persistent State: Remembers user's preference
CSS Class
<body class="sidebar-minimized">
  <!-- Minimized state activated -->
</body>

Responsive Sidebar Behavior

Desktop (â‰¥1024px)
  • Sidebar is visible by default
  • Can be minimized to icon-only mode
  • Fixed position, doesn't scroll with content
Tablet & Mobile (<1024px)
  • Sidebar is hidden by default (off-canvas)
  • Accessible via hamburger menu button
  • Overlays main content when opened
  • Can be closed with close button or backdrop click
Breakpoint
/* Tablet & Mobile */
@media (max-width: 1023px) {
  .sidenav-menu {
    position: fixed;
    left: -280px;
    transition: left 0.3s;
  }
  
  .sidenav-menu.show {
    left: 0;
  }
}

Sidebar Design Best Practices

  • check_circle Clear Hierarchy: Organize menu items logically with proper grouping
  • check_circle Active States: Clearly indicate the current page or section
  • check_circle Icons: Use consistent, recognizable icons for better usability
  • check_circle Scrollable: Ensure menu is scrollable when content exceeds viewport
  • check_circle Accessibility: Support keyboard navigation and screen readers
  • check_circle Performance: Lazy load sub-menus if the menu is very large

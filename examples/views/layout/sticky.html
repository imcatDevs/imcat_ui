Sticky Position Overview

The position: sticky CSS property creates elements that stick to the viewport when scrolling.

info Current Layout: This documentation uses sticky positioning for the topbar.
Basic Syntax
.sticky-element {
  position: sticky;
  top: 0;
  z-index: 100;
}

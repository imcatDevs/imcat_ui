Tabs - 수직 레이아웃

사이드바 스타일의 수직 탭

관리자 패널

대시보드

전체 통계와 최근 활동을 확인합니다.

주문 관리

모든 주문을 관리하고 추적합니다.

제품 관리

제품 카탈로그를 관리합니다.

고객 관리

고객 정보와 구매 이력을 확인합니다.

분석 리포트

매출, 트래픽, 전환율 등을 분석합니다.

설정 메뉴

프로필 설정

이름, 이메일, 프로필 사진을 변경합니다.

보안 설정

비밀번호, 2단계 인증을 관리합니다.

알림 설정

이메일, 푸시 알림을 설정합니다.

언어 설정

표시 언어와 시간대를 변경합니다.

사용법

기본 사용법

const Navigation = await IMCAT.use('navigation');

// Tabs 생성
const tabs = new Navigation.Tabs('#tabs_basic', {
  activeIndex: 0,
  orientation: 'horizontal',
  onChange: (index, tab, panel) => {
    console.log('탭 변경:', index);
  }
});

// 탭 선택
tabs.select(2);

// 정리
tabs.destroy();

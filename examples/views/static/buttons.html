Default Buttons

Use any of the available button classes to quickly create a styled button.

Button Rounded

Use .rounded-pill with button class to give pill-shaped rounded corners.

Button Outline

Use the .btn-outline-** classes to quickly create buttons with borders.

Outline Rounded

Use .btn-outline-** with .rounded-pill for rounded outline buttons.

Soft Buttons

Use the .btn-soft-** class for soft colored buttons.

Soft Rounded Buttons

Use .btn-soft-** with .rounded-pill for soft rounded buttons.

Button Sizes

Want larger or smaller buttons? Use .btn-lg or .btn-sm to adjust the button size.

Disabled Buttons

Add the disabled attribute to make buttons look inactive.

Icon Buttons

Add Material Icons inside buttons for icon buttons.

Button Groups

Group multiple buttons together by wrapping them with the .btn-group container.

Buttons 버튼

다양한 스타일과 크기의 버튼 컴포넌트입니다.

Default Buttons

기본 솔리드 스타일 버튼입니다.

코드 보기 
<button class="btn btn--primary">Primary</button>
<button class="btn btn--success">Success</button>

Rounded Buttons

둥근 모서리의 Pill 스타일 버튼입니다.

코드 보기 
<button class="btn btn--primary" style="border-radius: 50px;">Primary</button>

Outline Buttons

외곽선 스타일 버튼입니다.

코드 보기 
<button class="btn btn--outline-primary">Primary</button>

Outline Rounded

외곽선 + Pill 스타일 버튼입니다.

코드 보기 
<button class="btn btn--outline-primary" style="border-radius: 50px;">Primary</button>

Soft Buttons

연한 배경색의 Soft 스타일 버튼입니다.

코드 보기 
<button style="background-color: rgba(33, 150, 243, 0.1)...">Primary</button>

Ghost Buttons

투명 배경의 고스트 버튼입니다.

코드 보기 
<button class="btn btn--ghost">Ghost Button</button>

Button Sizes

다양한 크기의 버튼입니다.

Outline Sizes
코드 보기 
<button class="btn btn--primary btn--sm">Small</button>
<button class="btn btn--primary">Medium</button>
<button class="btn btn--primary btn--lg">Large</button>

Buttons with Icons

아이콘이 포함된 버튼입니다.

코드 보기 
<button class="btn btn--primary">
  <i class="material-icons-outlined">add</i>
  <span>Add Item</span>
</button>

Icon Only Buttons

아이콘만 있는 버튼입니다.

다양한 색상
코드 보기 
<button class="btn btn--primary btn--icon">
  <i class="material-icons-outlined">favorite</i>
</button>

Button States

버튼의 다양한 상태입니다.

Outline States
코드 보기 
<button class="btn btn--primary">Normal</button>
<button class="btn btn--primary" disabled>Disabled</button>

Button Tags

다양한 HTML 태그로 버튼을 만들 수 있습니다.

Link Button
코드 보기 
<a href="#" class="btn btn--primary">Link Button</a>
<button type="button">Button Element</button>

Block Buttons

전체 너비 버튼입니다.

코드 보기 
<button class="btn btn--primary btn--block">Block Button</button>

Button Group

버튼 그룹입니다.

Text Button Group
Icon Button Group
Sizes
코드 보기 
<div class="btn-group">
  <button class="btn btn--outline-primary">Left</button>
  <button class="btn btn--outline-primary">Middle</button>
  <button class="btn btn--outline-primary">Right</button>
</div>

Link Button

링크 스타일 버튼입니다.

Link as Anchor
Button Toolbar
코드 보기 
<button class="btn btn--link">Link Button</button>

Inline Code

텍스트 중간에 inline code를 삽입할 수 있습니다. 예를 들어 const greeting = 'Hello World';와 같이 사용합니다.

CSS 속성도 표시할 수 있습니다: display: flex, margin: 1rem

Basic Code Block

기본 코드 블록 (다크 테마)

function greet(name) {
  return `Hello, ${name}!`;
}

const message = greet('World');
console.log(message);

Python Example

main.py Python 
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# Print first 10 numbers
for i in range(10):
    print(fibonacci(i))

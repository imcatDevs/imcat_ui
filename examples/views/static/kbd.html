Basic Keyboard Keys

단일 키: Ctrl, Alt, Shift, Enter

키 조합: Ctrl + C (복사)

키 조합: Ctrl + V (붙여넣기)

키 조합: Ctrl + S (저장)

Dark Variant

다크 스타일: Ctrl + A

모두 선택: Ctrl + Shift + A

Light Variant

라이트 스타일: Ctrl + Z

되돌리기: Ctrl + Shift + Z

Keyboard Key Sizes

Small: Ctrl + C

Default: Ctrl + C

Large: Ctrl + C

Keyboard Key Groups

Ctrl + C
Ctrl + V

파일 작업:

검색:

Ctrl + F 찾기

Common Shortcuts

Windows / Linux
  • Ctrl + C - 복사
  • Ctrl + X - 잘라내기
  • Ctrl + V - 붙여넣기
  • Ctrl + Z - 실행 취소
  • Ctrl + Y - 다시 실행
  • Alt + F4 - 닫기
macOS
  • ⌘ Cmd + C - 복사
  • ⌘ Cmd + X - 잘라내기
  • ⌘ Cmd + V - 붙여넣기
  • ⌘ Cmd + Z - 실행 취소
  • ⌘ Cmd + ⇧ Shift + Z - 다시 실행
  • ⌘ Cmd + Q - 종료

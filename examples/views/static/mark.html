Basic Mark

Mark Color Variants

Primary: 중요한 정보

Success: 성공 메시지

Danger: 위험 경고

Warning: 주의 사항

Info: 정보 안내

Yellow: 노란색 강조

Green: 초록색 강조

Pink: 분홍색 강조

Purple: 보라색 강조

Highlight (Strong Emphasis)

Default: 매우 중요한 정보

Primary: 주요 정보

Success: 성공적으로 완료

Danger: 위험! 주의 필요

Warning: 경고: 확인 필요

Search Results

검색어: "프레임워크"

IMCAT UI는 현대적인 프레임워크입니다. 이 프레임워크는 다양한 컴포넌트를 제공합니다.

프레임워크 사용법은 매우 간단하며, 문서화가 잘 되어 있습니다.

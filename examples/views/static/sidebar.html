Basic Sidebar

Note: 실제 Sidebar는 IMCAT UI의 왼쪽 네비게이션에서 확인할 수 있습니다. 아래는 Sidebar의 HTML 구조 예시입니다. 

<aside class="sidebar">
  <div class="sidebar__header">
    <h3 class="sidebar__title">IMCAT UI</h3>
  </div>
  
  <div class="sidebar__body">
    <nav class="sidebar__nav">
      <ul>
        <li class="sidebar__item">
          <a href="#" class="sidebar__link sidebar__link--active">
            <i class="material-icons-outlined">home</i>
            <span>홈</span>
          </a>
        </li>
        <li class="sidebar__item">
          <a href="#" class="sidebar__link">
            <i class="material-icons-outlined">dashboard</i>
            <span>대시보드</span>
          </a>
        </li>
      </ul>
    </nav>
  </div>
</aside>

Sidebar Variants

Default Sidebar

기본 사이드바로 현재 애플리케이션에서 사용 중입니다.

Dark Sidebar

다크 모드 사이드바는 .sidebar--dark 클래스를 추가하여 사용합니다.

Collapsed Sidebar

축소된 사이드바는 .sidebar--collapsed 클래스를 추가하여 아이콘만 표시합니다.

Key Features

  • 고정 너비: 기본 260px 너비로 고정
  • 스크롤: 콘텐츠가 많을 경우 자동 스크롤
  • 반응형: 모바일에서는 토글 버튼으로 제어
  • 활성 상태: 현재 페이지 강조 표시
  • 아이콘 지원: Material Icons 통합

CSS Classes

Class Description
.sidebar 기본 사이드바 컨테이너
.sidebar__header 사이드바 헤더 영역
.sidebar__body 사이드바 본문 영역
.sidebar__nav 네비게이션 컨테이너
.sidebar__link 네비게이션 링크
.sidebar__link--active 활성 링크 상태
.sidebar--dark 다크 사이드바 변형
.sidebar--collapsed 축소된 사이드바

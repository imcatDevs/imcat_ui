info 개요

Background 유틸리티는 배경색, 테두리 스타일을 빠르게 적용할 수 있는 클래스를 제공합니다.

  • Subtle 배경색: 투명도가 적용된 부드러운 배경색
  • Border: 테두리 추가/제거 유틸리티
  • Border Color: 시맨틱 컬러 기반 테두리 색상
  • Directional Border: 특정 방향에만 테두리 적용

HTML 예제
<!-- Subtle Background -->
<div class="bg-primary-subtle p-3 rounded">

</div>

</div>

<!-- Combined -->

  Warning alert style
</div>

tips_and_updates Best Practices

  • Subtle 배경색: 중요하지 않은 정보를 부드럽게 강조할 때 사용하세요.
  • Border 조합: .border와 .border-{color}를 함께 사용하세요.
  • 시맨틱 컬러: 의미를 전달하는 적절한 컬러를 선택하세요 (success = 성공, danger = 오류 등).
  • Directional Border: 구분선이나 섹션 구분에 유용합니다.
  • !important 플래그: 모든 클래스는 !important를 사용하여 우선순위를 보장합니다.
  • CSS Variables: 테마 변경 시 자동으로 색상이 업데이트됩니다.

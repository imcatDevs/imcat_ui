info 개요

Display 유틸리티는 요소의 display 속성을 빠르게 변경할 수 있는 클래스를 제공합니다.

  • Display Types: none, block, inline, inline-block, flex
  • 반응형: 브레이크포인트별로 다른 display 값 적용 가능
  • 인쇄 제어: 인쇄 시 표시/숨김 제어

tips_and_updates Best Practices

  • 접근성: d-none 대신 스크린 리더를 위해 visually-hidden 클래스 사용을 고려하세요.
  • 반응형: 모바일 우선(Mobile First) 접근 방식을 사용하세요 (예: d-none d-md-block).
  • 성능: JavaScript 대신 CSS 클래스로 요소를 숨기는 것이 더 효율적입니다.
  • 조합: 여러 브레이크포인트 클래스를 조합하여 복잡한 반응형 동작을 구현할 수 있습니다.
  • 인쇄: 인쇄 시 특정 요소를 숨기려면 d-print-none 클래스를 사용하세요.

info 개요

Position 유틸리티는 요소의 위치와 z-index를 빠르게 제어할 수 있는 클래스를 제공합니다.

  • Position Types: static, relative, absolute, fixed, sticky
  • Edge Positioning: top, bottom, start(left), end(right)
  • Transform: translate-middle 중앙 정렬
  • Z-index: 레이어 순서 제어

tips_and_updates Best Practices

  • Relative + Absolute: absolute 요소는 relative 부모 컨테이너 내에서 사용하세요.
  • 중앙 정렬: top-50 start-50 translate-middle을 조합하여 완벽한 중앙 정렬을 구현하세요.
  • Z-index 계층: 일관된 z-index 시스템을 사용하여 레이어 충돌을 방지하세요.
  • Fixed 주의: position-fixed는 뷰포트 기준이므로 스크롤 영역에 주의하세요.
  • Sticky 브라우저 지원: position-sticky는 구형 브라우저에서 지원되지 않을 수 있습니다.
  • 접근성: absolute/fixed 요소가 컨텐츠를 가리지 않도록 주의하세요.
